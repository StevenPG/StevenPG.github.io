Memory Card Game

Memory Card Game was a project designed for CSC421 as a semester long project. The project required creating a working game of Memory with total creative control.

The project was implemented using the GWT framework and the Java for server data handling.

Online multiplayer is partially implemented with a lobby system that supports four players with the ability to chat simultaneously.

'Round The Corner


Round The Corner is a simple android application that served the purpose of learning how to use standard Android library functions like Views, Notifications, Location Services, Background Services, and multiple others.

The application takes inputs from the user and checks the distance between the device and the location found through geo-caching.

The user enters a location and a distance. When the device is within the input distance to the input location, a text message is automatically send to the input phone number.

There are simple binary save and load options that allow the user to save messages and numbers for their next use of the application.

Project Management App Web Service

This project was the back end web service for the Project Management App. Written in Python, the service retrieved data through HTTP verbs and made calls to an SQLite3 database on the same server.

The python application was built using the Flask microframework for web access, an Gunicorn as the application server.

KUSelfTour


KUSelfTour (AKA KUAR) is an android application created as part of the interdisciplinary course CSC354 ( Software Engineering). The app allows prospective and current students with an Android device to use their device's camera to learn information about campus buildings.

Pointing the device at buildings allows a user to select that building and have relevant information displayed.

Email Servlet

The mailer is a servlet designed to send an email when receiving data to my personal email directly. The application is deployed using Tomcat 8.0 and the 1.7 JRE.

The application uses the javax.mail library to handle mailing features. The war file can be directly deployed in tomcat and the application should be fully functional.

Remove Cores

This python script searches recursively through directories and removes any files with the name "core".

These types of files are usually generated in linux distribution when segmentation faults occur.

External IP API

This class is designed to allow a developer easy access to their machines external ip address. There is a single static method that will use Amazon Web Services to retrieve this caller's IP address.

Graph Theory Project

This project was the semester project for MAT369 Graph Theory.

The focus of my project was comparing the differences between the A* pathfinding algorithm and Dijkatra's pathfinding algorithm.

I wrote a research paper, along with giving a presentation in class discussing the differences between these algorithms. I also developed a simple web application demonstrating the strengths and weaknesses of these two algorithms.
