'Round The Corner

Round The Corner is an android application that allows a user to preset a contact, a message, an address, and a delivery distance.

Using GPS, the application detects when the device is within the delivery distance to the entered address and proceeds to send, hands free, the input message to the selected contact!

Perfect for letting someone know hands free when you are within a certain distance to their location.

Kutztown University Augmented Reality

KUAR is short for Kutztown University Augmented Reality. KUAR is an android application that was designed, tested, and implemented throughout the college course: CSC354WI, Software Engineering.

KUAR allows prospective students with an Android device to use their camera to learn information about campus buildings. Pointing the device at buildings allows a user to select that building and have relevant information displayed.

The primary objective of the course was to design an application completely from scratch while following the standards of a modified agile methodology.

I led the team of five Kutztown University students throughout the project. The team was comprised of myself, Justin, Lori and two CDE (Communication and Design) majors.

Each student was responsible for some portion of the application. The CDE students worked together to generate all designs and draft each prototype before each weekly meeting. Lori was in charge of documentation and some interface design work for the application UI. Justin's role was implementing a Google-Maps style UI that allowed the user to see their current location. My role was to design and implement location based augmented reality. Due to lacking or expensive libraries, my role expanded to design a limited API for augmented reality.

Java External IP Checker


The Java External IP Checker is a simple class containing only one static method. To use this method, the only requirement is importing the .java file into a project.

Java External IP Checker's static method retrieves the calling machine's external IP address by making a call to the public facing, free amazon web services API.

Libraries used:

  • java.io.BufferedReader
  • java.io.InputStreamReader
  • java.io.IOException
  • java.net.URL
  • java.net.MalformedURLException

If an issue occurs, an error code is returned. The meaning of each code is outlined in the source code of the class.

Usage: String ip = IpChecker.getIP();
