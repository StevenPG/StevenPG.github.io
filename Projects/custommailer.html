Java Custom Mailer My first Java EE project

Project Description

The Java Custom Mailer was originally written to be hosted on my home server and allow anyone to access my portfolio page, navigate to the #contact section, and open a form that would send data to the application.

Development went smoothly and the application fully functions using Tomcat 8.0 and is compiled using Java 1.7. The application uses a custom JSON reader that uses json.simple and a custom mailer object that utilizes javax.mail .

Due to timing and scope, I was unable to get this application to fully work on my local server. The application was nearly functional, but the core aspect, the emailing, was taking too long to run and causing the client to error out with an HTTP 500 error. After researching some security risks involved with mitigating this issue, I found that this was for the best. The application can be downloaded below and when the .war is deployed in a tomcat server, the application is fully functional.
