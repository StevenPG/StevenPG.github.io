Your-Reddit-Bot

This bot's primary purpose is to allow a user to select the settings they would like the bot to follow, and then create a bot that runs on a host machine (can be a personal computer or a server) against the specified settings. The primary means of communication is emailing. An example would be asking the bot to send an email for every text post that has "Star Wars" in the name. The primary means of communication may be updated to add more options in the future.

PhysCal

PhysCal is a past project between myself and an astro-physics student at Penn State University. It is an online calculation engine for physics questions. Originally intended to have multiple topics, the only currently implemented module is Kinematic equations. Due to scheduling issues and increasing school demands, the project has indefinitely been placed on hold.
