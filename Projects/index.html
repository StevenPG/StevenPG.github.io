Kutztown University Augmented Reality - (KUAR)
Android Application
Developed using Android Studio
KUAR is an android application that allows an android device user to use their device's camera to display the name of the building the camera is currently viewing. The user can then select this building and access multiple levels of information. Also included is a Google Maps-esque map that allows the user to select buildings based on their proximity and allows them to see their exact location while on campus.






NightBrowser
Java Application
Developed using Eclipse
The main purpose of this browser is to have a web browser that, unlike most software suites today which focus on bright white and "clean" looking background for their applications, will have a dark background and allow for use in darker environments, as well as for those who have a preference for darker backgrounds and do not have the customization options in their favorite applications.

Usage Statistics
Unix Command
Running this class places the usage statistics of all mounted devices on the calling machine into a text file in the same directory the script was run called usageStats.txt. This class is soon to be upgraded into a installation package that will alias the class as a command. Running only the command will display the information usually saved in usageStats.txt, while adding a -p to the call and the path will place the text file for reading. This was developed to allow a text file to be read by a webserver to actively display the current machine's storage information to a web browser dynamically.

Core Remove
Python Script
For use in Linux-based systems
A short python script that recursively navigates (linux compatible only) through a directory tree and deletes all 'core' files. These files can be created from segmentation faults and bus errors in C and C++ programs for example.
