Kutztown University Augmented Reality

Project Description

KUAR is short for Kutztown University Augmented Reality and is an android application that was designed, tested, and implemented throughout the college course: CSC354WI, Software Engineering.

The app allows prospective students with an Android device to use their camera to learn information about campus buildings. Pointing the device at buildings allows a user to select that building and have relevant information displayed.

The primary objective of the course was to design an application completely from scratch while following the standards of a modified agile methodology.

Project Details

  • I led the team of five Kutztown University students throughout the project. The team was comprised of myself, Justin, Lori and two CDE (Communication and Design) majors.
  • Roles:
    • My role was to design and implement location based augmented reality. Due to lacking or expensive libraries, my role expanded to design a limited API for augmented reality.
    • The CDE students worked together to generate all designs and draft each prototype before each weekly meeting.
      • Lori was in charge of documentation and some interface design work for the application UI.
    • Justin's role was implementing a Google-Maps style UI that allowed the user to see their current location.
