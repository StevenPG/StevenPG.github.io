Current projects

Your-Reddit-Bot

This bot's primary purpose is to allow a user to select the settings they would like the bot to follow, and then create a bot that runs on a host machine (can be a personal computer or a server) against the specified settings. The primary means of communication is emailing. An example would be asking the bot to send an email for every text post that has "Star Wars" in the name. The primary means of communication may be updated to add more options in the future.

KU Self Tour

This was the result of a joint class project between Kutztown Computer Science students and Kutztown Communication Design students. The resulting application is an augmented reality application that will help a prospective student to figure out which buildings are which and easily access important information regarding each building.

'Round The Corner

This android app allows a user to preset a contact and message that will be delivered when within a preset distance to a preset location. Allowing for handsfree notification of how far away from an address the device is at.

Java-Ip-Checker

A short Java library with exactly one static method call that allows for easy access to a caller's external IP address. Courtesy of Amazon's aws.

Command-Line-Games

This repo contains games created that can be played exclusively on a command line. They are recreations of classic boardgames and are single player.

Core-Remove

A short python script that recursively navigates (linux compatible only) through a directory tree and deletes all 'core' files. These files can be created from segmentation faults and bus errors in C and C++ programs for example.

Virtual-FishCenterLive

This project was created as a joke. FishCenterLive is a television series that airs on adult swim. This application attempts to replicate the fish tank and the fish starring in the series.

P2PJTalk

P2PJTalk is an application created as practice using Socket programming. This application allows two users on the same network (or a port fowarded alternate networks) to set up a Peer-to-Peer chat client between the two involved machines.

PhysCal

PhysCal is a past project between myself and an astro-physics student at Penn State University. It is an online calculation engine for physics questions. Originally intended to have multiple topics, the only currently implemented module is Kinematic equations. Due to scheduling issues and increasing school demands, the project has indefinitely been placed on hold.

SimpleCalculator

Simple Calculator is a simple Java program created using the Windowbuilder in eclipse and the Java Swing library. This project was to be used as a template and gain a deeper understanding of how to use git and github.

Get-Usage-Stats

This application quickly finds the important space statistics on the calling linux machine and compileds them into a text file.
